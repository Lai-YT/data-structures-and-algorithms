# Hash Table

An [open-addressed](https://en.wikipedia.org/wiki/Open_addressing), [double-hashed](https://en.wikipedia.org/wiki/Double_hashing) hash table in
C.

## Complexity

|Operation |Average |Worst case |
|----------|--------|-----------|
|Space     |O(*n*)  |O(*n*)     |
|search    |O(1)    |O(*n*)     |
|insert    |O(1)    |O(*n*)     |
|delete    |O(1)    |O(*n*)     |
